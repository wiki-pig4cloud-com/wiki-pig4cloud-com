## 服务添加 swagger 依赖
```xml
<!--接口文档-->
<dependency>
  <groupId>com.pig4cloud</groupId>
  <artifactId>pig-common-swagger</artifactId>
</dependency>
```

## 配置 EnablePigDoc 注解
<Note>
value 和 pig-gateway-dev.yml 里面路由前缀配置对应
</Note>

## 关闭 swagger 接口文档

nacos/application-dev.yml 配置关闭属性

```yaml
springdoc:
  api-docs:
    enabled: false
```
