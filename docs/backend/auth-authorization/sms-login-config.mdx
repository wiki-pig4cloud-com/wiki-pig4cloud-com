## 配置短信通道


+ 在 nacos 中配置如上参数即可

![](https://cdn.nlark.com/yuque/0/2021/png/283679/1640235829763-7717b76a-fea4-472e-a0cd-c1caab4bdd38.png)

```yaml
aliyun:
  sms:
    SMS_186401263:
      accessKeyId:  #阿里云AK
      accessKeySecret: #阿里云SK
      signName: #阿里云签名名
```
