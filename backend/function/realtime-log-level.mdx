---
title: "实时调整服务日志级别"
---

pig 默认的输出的日志级别是 INFO，生产中建议使用 ERROR 级别。

当我们遇到某些 bug 时，动态调整 log 输出日志非常有效，spring boot admin 提供了这样的功能。

### 启动 pig-monitor

+ 选中 PIG-UPMS 服务

![](/images/1599314988462-93e37aa0-35c9-4930-b1a7-cb7151780b97.png)

+ 选择 Loggers, 即可实现动态切换日志级别的需求

![](https://gitee.com/pig4cloud/oss/raw/master/2020-9/20200901153228.png)
