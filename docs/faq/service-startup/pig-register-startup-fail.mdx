## 问题说明
Pig Register 是基于 Nacos 的服务注册中心，未做额外封装。以下问题在使用原生 Nacos 时也会遇到。

### 1. 数据库加载错误 (DB Load Error)

<Warning>
如果遇到数据库加载错误，请按照以下步骤进行排查：
</Warning>

1. 检查 hosts 配置
   - 确认 pig-mysql 是否有效
   - 使用 ping 命令逐条验证 hosts 配置

2. 检查网络环境
   - 关闭本地 VPN 或代理软件
   - 这些软件可能会影响本地 hosts 解析

3. 检查数据库配置
   - 验证 pig-register/application.yml 中的配置信息
   - 确认 username、password、port 等配置是否正确

### 2. Web 登录问题

<Note>
如果遇到登录问题，请检查以下内容：
</Note>

1. 默认登录凭证
   - 用户名：nacos
   - 密码：nacos

2. 数据库初始化检查
   - 确认 pig_config 数据库是否正确初始化
   - 验证是否包含完整的 12 张表
   - 确保数据库版本为 5.7.8 或更高版本

<Card title="数据库版本要求" icon="info">
请确保使用 MySQL 5.7.8 或更高版本，否则可能会遇到兼容性问题。
</Card>
