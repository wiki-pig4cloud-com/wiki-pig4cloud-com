# Pig4Cloud 微服务开发平台

Pig4Cloud 是基于 Spring Cloud 的微服务快速开发平台，致力于提供稳定、高效、安全的企业级应用开发框架。

## 主要特性

- 基于 Spring Boot 和 Spring Cloud，遵循微服务设计原则
- 完善的安全体系和权限管理
- 内置丰富的企业级应用组件
- 支持分布式事务和服务治理
- 前后端分离架构，友好的开发体验
