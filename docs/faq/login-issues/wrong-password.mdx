## 错误提示

登录输入 admin / 123456 提示密码错误

![](/images/1615343649256-18f37b8e-c827-400e-ab0e-c46460d6c26c.png)

## 解决方案

1. 请在 Chrome 浏览器 F12 打开 network 查看报错
2. 如上报错请检查 auth、upms 模块是否已经正确启动，注意服务端报错信息
3. 确定密码解密是否成功 debugger 网关模块的如下 filter

![](/images/1641981467394-fe844a14-8a65-49f9-98d4-8e5708bd5a16.png)
