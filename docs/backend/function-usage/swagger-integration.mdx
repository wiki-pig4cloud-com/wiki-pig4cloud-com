## ①  服务添加 swagger  依赖
```xml
<!--接口文档-->
<dependency>
  <groupId>com.pig4cloud</groupId>
  <artifactId>pig-common-swagger</artifactId>
</dependency>
```

## 关闭 swagger 接口文档


+ nacos/application-dev.yml 配置关闭属性



```yaml
springdoc:
  api-docs:
    enabled: false
```
