## 解决方案

<Note>
请按照以下步骤检查并解决问题：
</Note>

1. 确认用的什么版本（前后端版本要保持一致）、什么架构 （微服务、单体）
2. 检查 Redis 服务是否正常运行
3. 检查 `.env` 文件中的 `VITE_IS_MICRO` 配置项是否与当前服务端架构匹配
4. 检查后台网关模块是否有错误日志
