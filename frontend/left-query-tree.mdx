## 基本用法

```html
<query-tree :query="state.queryList" @node-click="handleNodeClick">
</query-tree>
```

```typescript
const QueryTree = defineAsyncComponent(
  () => import("/@/components/QueryTree/index.vue")
);

const state = reactive({
  queryList: (name: String) => {
    return api({
      name: name,
    });
  },
});

const handleNodeClick = (e: any) => {};
```

## 插槽扩展

```html
<query-tree>
  <template #default="{ node, data }"> </template>
</query-tree>
```

## 属性说明

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| props | 树形结构的属性配置 | Object | `{ label: 'name', children: 'children', value: 'id' }` |
| placeholder | 输入框的占位符 | String | '' |
| loading | 是否显示加载状态 | Boolean | false |
| query | 包含查询方法的函数，必须返回 Promise 类型的数据，并在组件中被调用 | Function | - |
