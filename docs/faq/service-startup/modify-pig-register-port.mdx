## 问题
pig-register 模块是直接运行的 nacos-console 模块，不是标准的 springboot 启动过程**无法通过修改 application.yml 里面端口号 实现启动端口变更**。



## 解决方案


在 pig-register 启动 Main 方法增加



```java
System.setProperty("server.port","8080");
```
