# Pig4Cloud Wiki 文档结构

这是 Pig4Cloud 微服务快速开发平台的官方文档仓库。本文档使用 MDX 格式编写，并通过 Mintlify 进行构建和展示。

- 参与开发
- 学习资料

### 2. 部署运行
- 说明-版本分支
- 开发-快速开始
- 进阶-单体使用
- Linux-部署应用
- K8S-部署应用

### 3. 二次开发
- 创建微服务项目骨架
- 微调微服务配置管理
- 进阶-单体版二次开发

### 4. 后端开发
- 编程基础
- 微服务基础
- 网关功能
- 认证授权
- 功能使用

### 5. 前端开发
- 浏览器兼容性
- 前端配置项说明

- 国际化配置

### 6. 生产部署

### 7. 整合能力

### 8. 常见问题
- 提问须知
- 常见错误及解决方案

## 文档规范

1. 所有文档文件使用 MDX 格式编写
2. 每个文档顶部必须包含元数据区块：
   ```md
   ---
   title: "文档标题"
   ---
   ```
3. 文档图片存放在 `/images` 目录下
4. 文档结构由 `docs.json` 配置文件管理

## 贡献指南

如需贡献文档，请遵循以下步骤：
1. 按照相应分类在对应目录创建 MDX 文件
2. 确保文件顶部包含正确的元数据
3. 在 docs.json 中适当位置添加文档链接
