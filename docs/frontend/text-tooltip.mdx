## 提示组件

```html
<el-form-item label="初始浏览量" prop="visit">
  <template #label>浏览量<tip content="初始值" /></template>
</el-form-item>
```

## 属性说明

| 属性名称 | 描述 | 类型 | 默认值 |
| --- | --- | --- | --- |
| content | 内容 | String | - |
| placement | 放置位置 | String | 'top-start' |
