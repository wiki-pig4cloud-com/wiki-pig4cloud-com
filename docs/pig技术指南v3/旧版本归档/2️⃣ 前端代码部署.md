## ① 检查node版本
```shell
#  检查node版本 = 16
node -v 

# 若高版本node 可执行n 命令切换至 16 版本
n 16.17.1
```

## ② 下载前端代码


```shell
git clone https://gitee.com/log4j/pig-ui.git
```



## ③ 安装依赖


```shell
npm install  --registry=https://registry.npmmirror.com
```



## ④ 启动


```shell
npm run dev
```
