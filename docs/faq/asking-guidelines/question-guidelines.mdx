</Note>

2. 详细的各模块日志及截图；

    2. 后端日志：使用 IDEA 查看控制台日志，若为微服务，请提供网关、认证中心、业务微服务的详细日志
    3. 请提供完整的截图，不要只截取部分内容

3. 你做了什么修改、如何回显，请提供前后端的每步代码截图。
