## 常见问题

### 1. 运行提示缺少依赖

<Warning>
运行新模块提示缺少依赖报错，如下图报错
</Warning>

![缺少依赖报错](/images/1661477533668-37521260-69b3-4fa5-ab46-02501843a0a9.png)

### 2. 解决方案

- 检查是否**编译 PIG 微服务项目**  
- 检查创**建的项目和 PIG 项目版本**是否保持一致
- 检查 IDEA Maven 设置是否都是关联的目标仓库

![Maven 设置](/images/1661477769308-44a8e837-9bf7-406e-98d6-a32877cc5a7e.png)
