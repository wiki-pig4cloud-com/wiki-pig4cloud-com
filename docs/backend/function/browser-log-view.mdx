---
title: "配置 Logfile Viewer 日志挂载"
---

开启后可以在浏览器实时查看当前日志输出

### pig 默认没有开启 Logfile viewer

+ 配置 logback-spring.xml

可以直接复制 **pig-upms/pig-upms-biz/src/main/resources/logback-spring.xml** 文件，到新增服务的 resources 目录。

+ 默认路径会自动加载

![](/images/1619492131597-da4aec01-363b-4fd4-b386-834084f5f26a.png)

+ 其他路径日志挂载 配置 logging.file.name

在 bootstrap.yml 配置 logging.file 的配置即可开启 Logfile view

```yaml
# 配置 Logfile Viewer 日志挂载
logging:
  file:
    name: logs/${spring.application.name}/debug.log
```

### 启动 pig-monitor

+ 选中 目标服务
